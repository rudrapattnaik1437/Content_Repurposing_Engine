# 🎬 AttentionX - Automated Content Repurposing

## 🚀 Overview
AttentionX is a lightweight tool that converts long-form videos into short, engaging clips with captions. It helps creators, educators, and mentors quickly generate shareable content for platforms like Reels and TikTok.



## 💡 Problem
Long videos contain valuable insights, but users prefer short, engaging content. Manually finding and editing key moments is time-consuming.



## ✨ Solution
This app automates the process by:
- Splitting videos into short clips
- Adding captions
- Generating ready-to-share content instantly



## ⚙️ Features
- 📤 Upload video
- ✂️ Automatic clip generation
- 📝 Caption overlay
- ⚡ Fast processing (optimized for deployment)
- 🎥 Instant preview of clips



## 🛠 Tech Stack
- Python
- Streamlit
- MoviePy



## ▶️ How It Works
1. Upload a video  
2. Click "Generate Clips"  
3. The app splits the video into short segments  
4. Captions are added automatically  
5. View and download clips
